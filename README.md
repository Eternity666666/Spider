# Spider
新浪微博(Sina weibo)，百度搜索结果 爬虫



## 使用方法：

环境：python3

推荐使用虚拟环境：

```
cd weibo
chmod u+x env.sh
./env.sh
```

全局安装（不使用虚拟环境）：

 `pip3 install -r requirements.txt`



具体说明请见源码。
